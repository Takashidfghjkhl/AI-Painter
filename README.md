# Крокодил с ИИ

## Описание сайта
* Крокодил с ИИ на время
* Сохранение рисунков в формате `png`

## Примеры запуска оффлайн версии сайта
Для того чтобы запустить сайн оффлайн требуется:
1. Cкачать и разархивировать репозиторий.
2. Открыть в текстовом редакторе.
3. Запустить файл `index.html`
